# my-stock-analyzer
# 📊 顶级投行核心资产日线（Daily）量化扫描器

这是一个基于 Python + Streamlit 开发的股票日线趋势量化分析系统。专门用于监控半导体、科技股等核心资产的动能交叉、空间位置与量价异动。

## 🎯 核心量化策略体系
* **趋势动能**：采用 **9 EMA（短线快线）** 与 **24 SMA（月度慢线）** 组合。自动捕捉「金叉启动」、「死叉确立」以及「多/空趋势延展」。
* **空间定位**：融合 **布林带（Bollinger Bands）下轨** 与 **斐波那契（Fibonacci）38.2% / 61.8% 回撤线**，动态锁定强支撑位与阻力位。
* **量价雷达**：对比20日均量，自动识别 **🔥 放量（机构介入）** 与 **💤 缩量（盘整震荡）** 状态。
* **动能超买超卖**：集成 **RSI (14)** 指标，防范追高风险。
* **投行量化评分 (Score)**：结合上述所有维度进行综合打分，高分资产代表多头动能强劲且伴随突破。

## 📂 项目结构说明
* `app.py`：核心量化引擎与 Streamlit 前端交互主程序。
* `requirements.txt`：系统依赖的第三方环境包（yfinance, pandas 等）。
